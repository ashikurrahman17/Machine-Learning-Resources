It brings together lecture topics, theory notes, video tutorials, practice problems, and supplementary learning resources so that students can easily access everything required for the course without searching across multiple sources.

## 📌 Repository Purpose

This repository serves as a centralized learning hub for the **CSE 21 Batch Machine Learning course**. Its goal is to help students efficiently navigate the complete syllabus, access lecture-wise materials, and utilize curated learning resources—including theory, videos, and practice content—from a single, structured, and easy-to-use location.
